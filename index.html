About the Project

This project presents a smart attendance system powered by AI-based facial recognition. It eliminates the need for manual roll calls and biometric devices by automatically detecting and recording student attendance using AI cameras.

Key Features

  • Face recognition for accurate identification
  • Real-time attendance recording
  • Automated report generation
  • Data storage and analysis

Our Team

Alphonsa, Anand, Alwin Joshy, Anannya, Alwin Austin

How It Works

AI cameras are installed at classroom entrances. As students enter, their faces are detected and matched with the database. Attendance is marked automatically and securely stored in the cloud.
